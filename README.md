# CreditShield-AI-Explainable-Credit-Risk-Prediction-System
Credit Risk Default Prediction using Machine Learning

Project Overview

This project focuses on predicting the likelihood of customer loan default using Machine Learning techniques. The objective is to help financial institutions identify high-risk customers and make better lending decisions.

Dataset

The dataset contains customer demographic, financial, and credit-related information. Data preprocessing and feature engineering were performed to improve model performance.

Project Workflow

1. Data Loading and Exploration
2. Data Cleaning and Missing Value Treatment
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Handling Class Imbalance using SMOTE
6. Model Building and Evaluation
7. Model Interpretation using SHAP

Machine Learning Models Used

- Logistic Regression
- Random Forest Classifier
- XGBoost Classifier
- LightGBM Classifier
- CatBoost Classifier

Evaluation Metrics

- ROC-AUC Score
- Precision
- Recall
- F1-Score

Model Interpretation

SHAP (SHapley Additive exPlanations) was used to understand feature importance and explain model predictions.

Key Insights

- Customers with higher debt burdens are more likely to default.
- Repayment history plays a significant role in predicting credit risk.
- Financial behavior indicators strongly influence default probability

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- LightGBM
- CatBoost
- SHAP
- Matplotlib
- Seaborn

Conclusion

The project successfully developed machine learning models for credit risk prediction and provided interpretable insights that can support data-driven lending decisions.
